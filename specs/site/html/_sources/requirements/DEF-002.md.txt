# `DEF-002` Package Repository
{bdg-ref-primary}`repository <repository>`

A Package Repository is an on-disk directory structure that stores installed versions
of a single package.

The repository is structured as follows:
```
<package-name>/
  <version-1>/
  <version-2>/
  stable -> <version-1>   (symlink)
  experiment -> <version-2> (symlink)
```

The `stable` symlink points to the currently active version. The `experiment` symlink,
when present, points to a version being tested before promotion.

The repository never loads state into memory; all operations read directly from disk
to avoid divergence between in-memory and on-disk state.
