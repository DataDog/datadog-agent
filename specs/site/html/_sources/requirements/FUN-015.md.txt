# `FUN-015` Bootstrap installation
{bdg-ref-primary}`bootstrap <bootstrap>`

The installer shall be able to bootstrap itself from a minimal initial state.

The bootstrap process shall:
1. Download the installer package from the OCI registry.
2. Install the downloaded installer to the local repository.
3. Use the newly installed installer binary to install the remaining default packages.

This enables a single bootstrap command to go from no Datadog software installed
to a fully configured agent setup.
