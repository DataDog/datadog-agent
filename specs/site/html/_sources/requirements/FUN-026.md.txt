# `FUN-026` Change ddagentuser on existing install
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`msi <msi>`
{bdg-ref-primary}`user_account <user_account>`
> The installer shall be able to change the ddagentuser of an already installed Agent.
> 

The operator shall be able to change the user account under which the Agent services run
by specifying a new `DDAGENTUSER_NAME` (and optionally `DDAGENTUSER_PASSWORD`)
when running the installer on an existing installation.

After the change, the Agent services shall run under the new account with the correct
file and registry permissions.
