# `DEF-004` Extension
{bdg-ref-primary}`extension <extension>`

An Extension is a pluggable add-on that is attached to an installed package.

Extensions have their own install/remove lifecycle hooks but are tied to the parent
package's version. When the parent package is upgraded, extensions are saved before
the upgrade and restored after.

There is no concept of upgrading an extension independently of its parent package.
