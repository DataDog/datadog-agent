# `FUN-016` Remote-triggered operations
{bdg-ref-primary}`daemon <daemon>`
{bdg-ref-primary}`remote <remote>`

The fleet daemon shall listen for remote configuration updates from the Datadog backend
and execute package management operations accordingly.

Supported remote operations:
- Install a new package.
- Start, stop, or promote a package experiment.
- Start, stop, or promote a configuration experiment.

The daemon shall authenticate with the Datadog backend using the configured API key
and only accept operations from authorized sources.
