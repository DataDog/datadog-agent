# `FUN-022` Custom install directories
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`msi <msi>`
{bdg-ref-primary}`config <config>`
> The installer shall let the user chose a different directory for the Agent configuration and binaries.
> 

The Windows installer shall allow the operator to specify custom directories for:
- Agent configuration files (via `APPLICATIONDATADIRECTORY`).
- Agent binary files (via `PROJECTLOCATION`).

When custom directories are specified, the Agent shall use those directories
instead of the default locations.
