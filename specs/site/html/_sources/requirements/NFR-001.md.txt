# `NFR-001` Installer signing
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`security <security>`
> The installer shall be signed. A user only installs trusted, signed software on their machines.
> 

The Windows installer MSI package shall be cryptographically signed using a valid
code-signing certificate.

The signature shall be verifiable by the Windows operating system, so that users
and enterprise policies can confirm the installer's authenticity before execution.
