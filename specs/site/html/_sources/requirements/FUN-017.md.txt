# `FUN-017` Local HTTP API
{bdg-ref-primary}`daemon <daemon>`
{bdg-ref-primary}`api <api>`

The fleet daemon shall expose a local HTTP API for the CLI and other local clients
to trigger installer operations.

The API shall support:
- Installing a package by URL.
- Removing a package by name.
- Querying the state of installed packages.
- Starting, stopping, and promoting experiments.

The API shall only accept connections from the local machine.
