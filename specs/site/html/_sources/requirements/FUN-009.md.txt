# `FUN-009` Promote or rollback config experiment
{bdg-ref-primary}`config <config>`
{bdg-ref-primary}`experiment <experiment>`

The operator shall be able to promote a config experiment, making the patched configuration
permanent, or remove the config experiment, reverting to the previous configuration.

When promoting:
1. The patched configuration becomes the new base configuration.
2. The package's PostPromoteConfigExperiment hook is executed.

When reverting:
1. The previous configuration is restored.
2. The package's PreStopConfigExperiment hook is executed.
