# `FUN-018` Setup flavors
{bdg-ref-primary}`setup <setup>`

The installer shall support multiple setup flavors, each defining a specific combination
of packages and configuration for a particular use case.

The following flavors shall be supported:
- "default": Standard Datadog Agent installation.
- "APM SSI": Standalone APM Single Step Instrumentation setup.
- "databricks": Data job management for Databricks.
- "emr": Data job management for Amazon EMR.
- "dataproc": Data job management for Google Dataproc.

Each flavor determines which packages are installed and what configuration is applied.
Specifying an unknown flavor shall result in an error.
