# `FUN-006` Query package state
{bdg-ref-primary}`lifecycle <lifecycle>`
{bdg-ref-primary}`query <query>`

The operator shall be able to query the current state of any installed package.

The response shall include:
- Whether the package is installed.
- The stable version identifier (if installed).
- The experiment version identifier (if an experiment is active).

The operator shall also be able to query the state of all installed packages at once.
