# `FUN-003` Start an experiment upgrade
{bdg-ref-primary}`lifecycle <lifecycle>`
{bdg-ref-primary}`experiment <experiment>`

The operator shall be able to start an experiment that installs a new version of a package
alongside the current stable version.

The installer shall:
1. Execute the current stable version's PreStartExperiment hook.
2. Download and extract the new version to a versioned directory. The stable version's files
   are kept intact.
3. Set the "experiment" symlink to point to the new version.
4. Execute the new version's PostStartExperiment hook.

After starting an experiment, the package state shall show both a stable and an experiment version.
