# `DEF-001` OCI Package
{bdg-ref-primary}`oci <oci>`

An OCI Package is a Datadog software package distributed as an OCI (Open Container Initiative)
container image from a registry.

Each OCI image carries annotations that identify the package:
- `com.datadoghq.package.name`: The package name (e.g. `datadog-agent`).
- `com.datadoghq.package.version`: The package version.
- `com.datadoghq.package.size`: The package size.

The image contains a single layer in `application/vnd.datadog.package.layer.v1.tar+zstd`
media type, which is a zstd-compressed tar archive of the package contents.
