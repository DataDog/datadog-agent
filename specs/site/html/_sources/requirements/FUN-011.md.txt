# `FUN-011` Persist extensions across upgrades
{bdg-ref-primary}`extension <extension>`
{bdg-ref-primary}`lifecycle <lifecycle>`

Extensions shall be preserved across package upgrades.

When a package is upgraded, the installer shall:
1. Save the list of installed extensions and their state before the upgrade.
2. Perform the package upgrade.
3. Restore the previously installed extensions on the new package version.

The operator shall not need to manually reinstall extensions after upgrading a package.
