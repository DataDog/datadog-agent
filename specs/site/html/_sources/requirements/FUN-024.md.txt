# `FUN-024` Prevent downgrades
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`msi <msi>`
{bdg-ref-primary}`upgrade <upgrade>`
> The installer shall not allow downgrading the Agent installation.
> 

The Windows installer shall reject attempts to install an older version of the Agent
over a newer one (e.g. Agent 7 to Agent 6, or Agent 7.55 to Agent 7.50).

The installer shall display an error message indicating that downgrades are not supported.
