# `T001` Bootstrap from clean state installs the installer and default packages

Validates the minimal happy path: on a machine with no Datadog software,
running the bootstrap command downloads the installer, installs it,
and then installs the default packages resolved from the environment.

## Initial state
- No Datadog software is installed on the host.
- `DD_API_KEY` is set to a valid API key.
- `DD_SITE` is set (or defaults to datadoghq.com).
- The OCI registry is reachable.


## Procedure
### Step 1
Run the bootstrap command.
```console
$ datadog-installer bootstrap
```
### Step 2
Verify the installer package is present in the local repository.
```console
$ datadog-installer is-installed datadog-installer; echo $?
0
```
### Step 3
Verify the default packages (as resolved by the environment) are installed.
```console
$ datadog-installer is-installed datadog-agent; echo $?
0
```
