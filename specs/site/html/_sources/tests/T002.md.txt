## Initial state
- No Datadog software is installed on the host.
- `DD_API_KEY` is set.
- `DD_INSTALLER_REGISTRY_URL` is set to a custom registry URL.
- The custom registry hosts the installer image.


## Procedure
### Step 1
Run the bootstrap command with the custom registry configured.
```console
$ DD_INSTALLER_REGISTRY_URL=https://custom.registry.example datadog-installer bootstrap
```
### Step 2
Verify the installer package is installed.
```console
$ datadog-installer is-installed datadog-installer; echo $?
0
```
