## Initial state
- The installer is bootstrapped and functional.
- `DD_API_KEY` is set.
- The OCI registry is reachable.
- The target package is not yet installed.


## Procedure
### Step 1
Install a package by providing its OCI URL.
```console
$ datadog-installer install oci://gcr.io/datadoghq/agent-package:latest
```
### Step 2
Verify the package is installed.
```console
$ datadog-installer is-installed datadog-agent; echo $?
0
```
