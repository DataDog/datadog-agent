## Initial state

[`T005` Install a single package from OCI registry](/tests/T005)
## Procedure
### Step 1
Re-install the same package at the same version.
```console
$ datadog-installer install oci://gcr.io/datadoghq/agent-package:latest; echo $?
0
```
### Step 2
Verify the package is still installed.
```console
$ datadog-installer is-installed datadog-agent; echo $?
0
```
