## Initial state
The expected state is the final state of
[`T001` Bootstrap from clean state installs the installer and default packages](/tests/T001)
## Procedure
### Step 1
Run the bootstrap command again on the already-bootstrapped host.
```console
$ datadog-installer bootstrap; echo $?
0
```
### Step 2
Verify all previously installed packages are still present.
```console
$ datadog-installer is-installed datadog-installer && datadog-installer is-installed datadog-agent; echo $?
0
```
