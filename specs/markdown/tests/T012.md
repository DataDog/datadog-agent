# `T012` Remove then reinstall a package

Validates that a package can be cleanly reinstalled after removal,
confirming that removal does not leave behind state that prevents
a fresh installation.

## Initial state
The expected state is the final state of
[`T010` Remove an installed package](/tests/T010)
## Procedure
### Step 1
Reinstall the previously removed package.
```console
$ datadog-installer install oci://gcr.io/datadoghq/agent-package:latest; echo $?
0
```
### Step 2
Verify the package is installed again.
```console
$ datadog-installer is-installed datadog-agent; echo $?
0
```
