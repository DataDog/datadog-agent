## Procedure

Force-install the same package.
```console
$ datadog-installer install --force oci://gcr.io/datadoghq/agent-package:latest; echo $?

```
### Step 2
Verify the package is still installed.
```console
$ datadog-installer is-installed datadog-agent; echo $?
0
```
