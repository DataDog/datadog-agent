## Initial state
- No Datadog software is installed on the host.
- `DD_API_KEY` is set.
- `DD_INSTALLER_REGISTRY_URL` points to an unreachable host.


## Procedure
### Step 1
Run the bootstrap command against the unreachable registry.
```console
$ DD_INSTALLER_REGISTRY_URL=https://unreachable.invalid datadog-installer bootstrap; echo $?
1
```
### Step 2
Verify no installer package was left behind.
```console
$ datadog-installer is-installed datadog-installer; echo $?
1
```
