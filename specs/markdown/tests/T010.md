## Initial state

[`T005` Install a single package from OCI registry](/tests/T005)
## Procedure
### Step 1
Remove the installed package.
```console
$ datadog-installer remove datadog-agent; echo $?
0
```
### Step 2
Verify the package is no longer installed.
```console
$ datadog-installer is-installed datadog-agent; echo $?
1
```
