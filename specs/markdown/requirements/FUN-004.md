# `FUN-004` Promote an experiment to stable
{bdg-ref-primary}`lifecycle <lifecycle>`
{bdg-ref-primary}`experiment <experiment>`

The operator shall be able to promote a running experiment, making the experiment version
the new stable version.

The installer shall:
1. Execute the experiment version's PrePromoteExperiment hook.
2. Remove the old stable version's files from disk.
3. Move the "stable" symlink to point to the experiment version.
4. Remove the "experiment" symlink.
5. Execute the old stable version's PostPromoteExperiment hook.

After promotion, the package state shall show the promoted version as stable
with no active experiment.
