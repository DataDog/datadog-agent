# `FUN-007` Garbage collect unused versions
{bdg-ref-primary}`lifecycle <lifecycle>`
{bdg-ref-primary}`maintenance <maintenance>`

The installer shall be able to garbage-collect package versions that are no longer referenced
by either the stable or experiment symlinks.

After garbage collection, only the directories pointed to by the stable and experiment symlinks
(if any) shall remain on disk for each package.
