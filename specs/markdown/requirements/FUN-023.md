# `FUN-023` Major and minor version upgrades
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`msi <msi>`
{bdg-ref-primary}`upgrade <upgrade>`
> The installer shall be able to upgrade an Agent to a new major version.
> The installer shall be able to upgrade an Agent with the same major version but with a lower minor version.
> 

The Windows installer shall support the following upgrade paths:
- Agent 5 to Agent 6.
- Agent 5 to Agent 7.
- Agent 6 to Agent 7.
- Agent 6.x to a newer Agent 6.y (where y > x).
- Agent 7.x to a newer Agent 7.y (where y > x).

After a successful upgrade, the Agent shall be in a working state running the new version.
