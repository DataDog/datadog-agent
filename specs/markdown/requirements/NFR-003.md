# `NFR-003` Clean state on failure
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`resilience <resilience>`
> When the installer fails to install, it shall leave the target system in a clean state.
> 

When the installer fails during a fresh installation, it shall leave the target system
in a clean state with no partial installation artifacts.

Specifically:
- The Agent binary directory shall not exist.
- The Agent configuration directory shall be empty (except for any pre-existing YAML files).
- No orphan services shall be registered.
- No orphan user accounts shall be created.
