# `FUN-013` Default package resolution
{bdg-ref-primary}`config <config>`
{bdg-ref-primary}`packages <packages>`

The installer shall resolve which packages to install by default based on the environment.

The resolution shall consider:
- Whether each package is marked as released (generally available).
- Whether the package is released for the configured site.
- Whether remote updates are enabled (some packages require this).
- Package-specific conditions (e.g. APM injection requires APM to be enabled).
- Explicit overrides via `DD_INSTALLER_DEFAULT_PKG_INSTALL`.

The operator shall be able to override the default resolution to force-include or
force-exclude specific packages.
