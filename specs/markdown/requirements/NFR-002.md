# `NFR-002` Least privilege
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`security <security>`
> The Agent services shall run with the minimum amount of privileges necessary.
> 

The Agent services shall run under a user account with the minimum privileges
necessary for their operation.

The service account shall not have administrative privileges unless explicitly required.
File, directory, and registry permissions shall be restricted to only what the Agent needs.
