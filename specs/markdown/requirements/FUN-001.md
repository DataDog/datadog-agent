# `FUN-001` Install a package from OCI registry
{bdg-ref-primary}`lifecycle <lifecycle>`
{bdg-ref-primary}`oci <oci>`

The operator shall be able to install a package by providing its OCI registry URL.

The installer shall:
1. Download the OCI image from the specified registry, authenticating as configured.
2. Extract the package contents to a versioned directory in the local package repository.
3. Set the "stable" symlink to point to the newly installed version.
4. Execute the package's PostInstall hook.

After a successful install, the package shall appear as installed when querying package state,
and the stable version shall match the version that was just installed.

If the download or extraction fails, no changes shall be made to the repository state.
