# `NFR-005` Telemetry
{bdg-ref-primary}`observability <observability>`

The installer shall emit OpenTelemetry traces for all operations.

Each installer operation (install, remove, experiment start/stop/promote, config experiment,
bootstrap, setup) shall produce a trace span that can be used for monitoring and debugging.
