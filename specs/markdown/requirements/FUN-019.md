# `FUN-019` Repair damaged installation
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`msi <msi>`
{bdg-ref-primary}`repair <repair>`
> If the Agent installation is damaged, running the installer shall repair it.
> 

The operator shall be able to repair a damaged Agent installation by re-running the installer.

If the Agent service user account has a password, the operator should provide it
to ensure the services can be repaired.

After a successful repair, the Agent shall be in a working state with all services running.
