# `ARC-002` Dual-state repository with symlinks
{bdg-ref-primary}`repository <repository>`

Each installed package maintains two symlinks on disk: `stable` and `experiment`,
pointing to versioned directories.

This architecture decision enables:
- Atomic version switching by updating a single symlink.
- Safe rollback by keeping both versions on disk simultaneously.
- No in-memory state that could diverge from the filesystem.
- Tolerance of process crashes mid-operation (the symlink is either updated or not).
