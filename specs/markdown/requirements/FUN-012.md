# `FUN-012` Configure via environment variables
{bdg-ref-primary}`config <config>`
{bdg-ref-primary}`environment <environment>`

The installer shall read its configuration from environment variables.

The following environment variables shall be supported:
- `DD_API_KEY`: The Datadog API key.
- `DD_SITE`: The Datadog site (default: datadoghq.com).
- `DD_REMOTE_UPDATES`: Enable remote update support (boolean).
- `DD_INSTALLER_REGISTRY_URL`: Override the OCI registry URL.
- `DD_INSTALLER_REGISTRY_AUTH`: Registry authentication method (docker, gcr, password).
- `DD_INSTALLER_MIRROR`: Package mirror URL.
- `DD_PROXY_HTTP`, `DD_PROXY_HTTPS`, `DD_PROXY_NO_PROXY`: Proxy configuration.
- `DD_APM_INSTRUMENTATION_ENABLED`: Enable APM instrumentation.
- `DD_APM_INSTRUMENTATION_LIBRARIES`: APM library selection.

When an environment variable is not set, a sensible default shall be used.
