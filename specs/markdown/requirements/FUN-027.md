# `FUN-027` Create install_info file
{bdg-ref-primary}`windows <windows>`
{bdg-ref-primary}`msi <msi>`
{bdg-ref-primary}`telemetry <telemetry>`
> The installer shall create the install_info file, describing how the Agent was installed.
> 

The installer shall create an `install_info` file in the Agent configuration directory
that describes how the Agent was installed.

For GUI installs, the file shall contain:
- `tool`: `windows_msi_gui`
- `tool_version`: `windows_msi_next_gen`
- `installer_version`: the installer version string.

For CLI (quiet) installs, the file shall contain:
- `tool`: `windows_msi_quiet`
- `tool_version`: `windows_msi_next_gen`
- `installer_version`: the installer version string.
