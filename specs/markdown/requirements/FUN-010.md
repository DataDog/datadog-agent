# `FUN-010` Install and remove extensions
{bdg-ref-primary}`extension <extension>`

The operator shall be able to install or remove extensions for an installed package.

When installing an extension:
1. The package's PreInstallExtension hook is executed.
2. The extension files are written to disk.
3. The package's PostInstallExtension hook is executed.

When removing an extension:
1. The package's PreRemoveExtension hook is executed.
2. The extension files are removed from disk.
