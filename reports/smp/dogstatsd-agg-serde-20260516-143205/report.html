DogStatsD Aggregator + Serializer SMP Report

Local SMP run directory: reports/smp/dogstatsd-agg-serde-20260516-143205/

Current verdict: Stage A-Q are complete locally, and the post-Stage-Q raw-ring telemetry/honesty matrix is complete. Stage I was negative because the columnar parser path computed an unused debug projection for every sample. Stage J removed that artifact and turned the narrow columnar v3 slice into a +13.57% high-rate throughput win. Stage L showed a byte-bounded ingress log can trade unconstrained overload admission for explicit backpressure. Stage M's raw UDS ring removed packet-pool backlog entirely. Stage N/O's compact raw ring plus batch drains remain the best bounded high-rate design. Stage P falsified the simple no-copy direct-reservation idea. Stage Q adds native columnar-v3 payload construction and a v2/direct-series bridge. Three-replicate honesty gates still show positive UDS origin-off v2/v3 throughput, but high-rate wins are explicitly bounded by raw-ring backpressure; UDP/origin-on are raw-disabled fallback checks.

Current value read: Stage I had not actually collapsed identity work: it added an unused debug-key and display-tags projection on every parser hot-path sample. Stage J removes that artifact and shows the narrow columnar design can beat direct metric rows. Stage L shows the large packet queue can be replaced by explicit ingress-log backpressure. Stage M shows a raw UDS ring can eliminate packet-pool backlog. Stage N shows byte-compact retained records matter. Stage O shows batched worker drains are a real but smaller high-rate ceiling step. Stage P shows one-max-slot direct reservation is not the next ceiling. Stage Q completes direct native v3/v2 payload construction but is not a large standalone win over Stage O. Stage R makes proof telemetry and naive descriptor interning explicit costs rather than hidden defaults. Stage S shows direct v3 serializer live heap is not the main RSS culprit and removes low-risk row-copy escapes. Stage T confirms parser/string interning is a real remaining bottleneck; SLRU reset avoidance is safe and exact tagset dictionaries can substantially reduce backpressure on repeated-tagset workloads. Stage N/O plus Stage Q/R/S/T remains the best tested combined design; post-telemetry honesty gates narrow the claim to UDS datagram/origin-off and make high-rate backpressure visible.
Stage E/F/G/H/I/J/L/M/N/O/P/Q/R/T caveat: this is intentionally unsafe/local and enabled through env gates. Stage E bypasses iterable channel traversal. Stage F emits DogStatsD time-sampler series as metrics.SerieRow. Stage G bypasses scalar *metrics.Serie materialization. Stage H bypasses grouping/dedup and is intentionally not wire-equivalent. Stage I/J/Q/R are v3-only/metric-oriented and fall back to legacy paths for unsupported samples/checks/sketches/events, except the explicit skip-legacy upper-bound probe and the Stage Q direct-series/v2 bridge. Stage L changes overload/backpressure behavior and is only a first-pass packet-batch ingress log. Stage M/N/O/P raw modes are UDS-datagram-only and currently exclude UDP, stream sockets, named pipes, statsd forwarding, and origin detection; the origin-on and UDP probes intentionally ran with raw ingress disabled. Stage R changes measurement cost shape by making proof shadows optional and descriptor interning opt-in. Stage T's exact tagset cache remains opt-in until admission/memory behavior is validated on mostly-unique/adversarial tagsets.
